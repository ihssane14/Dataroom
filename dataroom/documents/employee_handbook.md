### Section 1.1: Security Requirements
- All employees must complete annual security training
- Multi-factor authentication is mandatory for system access
- Sensitive documents must be encrypted before transmission
- Report security incidents to infosec@corporate.com

### Section 2.3: Data Classification
- PUBLIC: Information approved for external distribution
- INTERNAL: Company information not for public release
- CONFIDENTIAL: Business-sensitive information
- RESTRICTED: Highly sensitive competitive data
